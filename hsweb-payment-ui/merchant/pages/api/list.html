属性 内容

请求参数

请求参数 属性名 必填 说明
请求ID requestId 是 请求ID由商户生成16位随机数。
商户ID merchantId 是 系统给定的商户号
订单号 orderId 是 商户自己的订单号
交易金额 amount 是 交易金额，单位：分
产品唯一标识 productId 是 产品唯一标识（商户自己的产品ID）
产品名称 productName 是 产品名称,例:话费充值
异步通知地址 notifyUrl 否 异步通知地址,不能有参数，接口返回：success则通知成功，否则将阶梯性重试。
跳转地址 returnUrl 否 用户支付完成后跳转地址，只有部分渠道有效。
支付渠道 channel 是 支付渠道。例:alipay（参考【渠道标志】）
渠道特殊参数 extraParam 否 无可不填写，例：{"type","WEB"},如果是微信支付，则需要传IP地址,例如：{"spbillCreateIp":"192.168.1.1"}
签名 sign 是 签名算法看文档
